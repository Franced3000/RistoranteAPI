## RistoranteAPI
Back-end totalmente scritto in Typescript e Express.js.
### istruzioni
1. per avviare il progetto creare il file .env come da esempio:
DB_HOST=xxxxxx
DB_PORT=xxxxxx
DB_DATABASE=xxxxx
DB_USERNAME=xxxxx
DB_PASSWORD=xxxxx
DB_DRIVER=xxxxx
PORT=xxxxx
2. installare i pacchetti con npm:
npm install
3. avvia il progetto
npm run start
